# 🎮 훈민정음 초성 게임

**두 글자 초성으로 즐기는 실시간 단어 대결 게임!** Socket.io 기반
멀티플레이 턴제 게임으로, 여러 명이 같은 방에 들어와 초성 단어 대결을 할
수 있습니다.

👉 **배포 주소:**\
https://hunmin-quiz.onrender.com

------------------------------------------------------------------------

## 📌 주요 기능

### 🏠 로비 & 방 시스템

-   닉네임 입력 후 **방 생성 / 방 코드로 입장** 가능
-   최대 **8명**까지 동일한 방에서 플레이
-   방장은 게임 시작 권한 보유
-   방장 이탈 시 자동으로 다음 사람에게 방장 권한 이전

### 🎮 게임 규칙

-   한 라운드는 **하나의 초성(예: ㄱㅈ)** 으로 진행
-   플레이어는 **순서대로 턴을 가지고**, 2글자 단어 입력
-   단어 조건:
    1.  초성 두 글자와 정확히 일치
    2.  표준국어대사전 API에서 검증되는 실제 단어
    3.  이전에 사용된 단어는 사용 불가

### ⏱ 턴 시간 제한 & 점수

-   기본 10초 → 사이클이 반복될수록 점점 짧아짐
-   남은 시간 비율에 따라 **최대 +100점 획득**
-   시간 초과 시 **-100점 패널티 + 라운드 종료**

### 📚 UI 기능

-   실시간 **라운드 로그 표시**
-   정답 단어가 **카드 형태로 누적 표시**
-   카드 Hover 시 **전체 뜻 툴팁 제공**
-   내 턴일 때 시각적인 강조효과
-   정답 시 **토스트 애니메이션 출력**

### 🏆 게임 종료 화면

-   전체 플레이어 점수 정렬
-   🥇 1등에게 왕관 아이콘 표시
-   게임 재시작 / 로비 이동 가능

## 🔧 로컬 실행 방법

### 1. 레포 클론

``` bash
git clone https://github.com/soominn/hunmin-quiz.git
cd hunmin-quiz
```

### 2. 패키지 설치

``` bash
npm install
```

### 3. 환경변수 설정

루트에 `.env` 파일 생성:

    STD_KO_DICT_KEY=YOUR_STD_DICT_API_KEY

### 4. 개발 서버 실행

``` bash
npm start
```

👉 http://localhost:3000

1.  GitHub에 Push
2.  Render → **New Web Service**
3.  Build Command:\
    npm install
4.  Start Command:\
    npm start
5.  환경 변수 입력
6.  배포 완료 후 자동 URL 생성
